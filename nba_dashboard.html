Arquitetura do Pipeline
ETL Steps — Fluxo de Dados
01
EXTRACT extract.py
Coleta dados via nba_api: player game logs, league leaders, team results. Rate-limit safe com delay automático.
02
CLEAN clean.py
Padroniza colunas, converte tipos, remove duplicatas, filtra jogos sem minutos jogados.
03
TRANSFORM transform.py
Feature engineering: Impact Score, True Shooting%, Game Score, rolling averages (5/10j), Z-score outlier detection.
04
LOAD load.py
Upsert no PostgreSQL via SQLAlchemy. Sem duplicatas mesmo em múltiplas execuções.
05
DASHBOARD generate_dashboard.py
Gera este HTML com dados reais do banco. Atualize rodando o script novamente.
